Halo! Saya Fadlan Achmad Frizal, seorang Web Developer, Network Operation Center Staff dan Social Media Staff. Saya tinggal di Kemanggisan, Palmerah, Jakarta Barat. Saya mulai belajar pengembangan web saat sekolah menengah kejuruan, dan sejak itu saya tertarik untuk membuat dan mengembangkan website statis dan dinamis. Saat ini saya sedang berkuliah di Universitas Esa Unggul Jakarta di jurusan Sistem Informasi. Saya melakukan monitoring jaringan saat saya magang di PT Mitra Teknologi Andalan Utama, Selain itu, saya juga mengelola media sosial saat saya magang di Media Indonesia

Beberapa keterampilan yang saya kuasai antara lain HTML, CSS, Bootstrap, PHP, Laravel, MySQL, C++, Javascript, dan Java.

Di bidang jaringan, saya biasa menggunakan Polestar, SD-WAN VManage, Putty, Unifi, dan Winbox. Saya juga menjadi Helpdesk seperti menerima aduan gangguan melalui whatsapp, membuat tiket gangguan dan email terkait laporan gangguan.

Di bidang media sosial, saya biasa menggunakan Canva, CapCut untuk mengedit. Lalu saya bisa mengoperasi kan kamera, mikrofon, lampu tungsten dan LED untuk keperluan streaming maupun shooting video. Terakhir saya pernah menjadi pewawancara dalam program TrackInMi.
